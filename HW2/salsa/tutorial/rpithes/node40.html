Message Properties

SALSA provides three message properties that could be used with message sending: priority, delay, waitfor, and delayWaitfor. The syntax used to assign to a message a given property is the following, where <property name> can be either priority, delay, waitfor, and delayWaitfor:

actor<-myMessage:<property name>
