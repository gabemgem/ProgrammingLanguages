Beyond the Actor Model

While SALSA supports the actor model, it goes further in providing linguistic abstractions for common coordination patterns in concurrent and distributed applications. For concurrency, it provides token passing continuations, join blocks, first-class continuations, named tokens, and message properties. For distribution, it provides universal naming abstractions, location-transparent remote communication, and migration support. Furthermore, SALSA provides automatic local and distributed garbage collection.
