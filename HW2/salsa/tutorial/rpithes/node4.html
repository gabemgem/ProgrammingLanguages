Structure of the Tutorial

This tutorial covers basic concepts of SALSA and illustrates its concurrency and distribution models through several examples. Chapter introduces the actor model and how SALSA supports it. Chapter introduces concurrent programming in SALSA, including token-passing continuations, join blocks, and first-class continuations.Chapter discusses SALSA's support for distributed computing including asynchronous message sending, universal naming, and migration. Chapter introduces several advanced coordination constructs and how they can be coded in SALSA. Chapter defines actor garbage and explains how automatic actor garbage collection works in SALSA. Appendix introduces how to specify name server options and how to run applications with different system properties. Appendix provides debugging tips for SALSA programs. Appendix provides brief descriptions of SALSA example programs. Appendix lists the SALSA grammar.
