Input/Output (I/O) Actor Access

SALSA provides three kinds of I/O actors to support asynchronous I/O accesses. One of them is an input service (standardInput), and two are output services (standardOutput and standardError). Since they are actors, only asynchronous accesses are possible.

standardOutput provides the following message handlers:

  • print(boolean p)
  • print(byte p)
  • print(char p)
  • print(double p)
  • print(float p)
  • print(int p)
  • print(long p)
  • print(Object p)
  • print(short p)
  • println(boolean p)
  • println(byte p)
  • println(char p)
  • println(double p)
  • println(float p)
  • println(int p)
  • println(long p)
  • println(Object p)
  • println(short p)
  • println()
standardError provides the following message handlers:
  • print(boolean p)
  • print(byte p)
  • print(char p)
  • print(double p)
  • print(float p)
  • print(int p)
  • print(long p)
  • print(Object p)
  • print(short p)
  • println(boolean p)
  • println(byte p)
  • println(char p)
  • println(double p)
  • println(float p)
  • println(int p)
  • println(long p)
  • println(Object p)
  • println(short p)
  • println()
standardInput provides only one message handler in current SALSA release:
  • String readLine()
